Table of contents

## Introduction

You can fetch a single node or multiple nodes of the same type using a
simple object query.

## Fetch list of objects

**Example:** Fetch a list of authors:

<div class="graphiql" view_only="" query="query {
  authors {
    id
    name
  }
}
:response:
{
  &quot;data&quot;: {
    &quot;authors&quot;: [
      {
        &quot;id&quot;: 1,
        &quot;name&quot;: &quot;Justin&quot;
      },
      {
        &quot;id&quot;: 2,
        &quot;name&quot;: &quot;Beltran&quot;
      },
      {
        &quot;id&quot;: 3,
        &quot;name&quot;: &quot;Sidney&quot;
      },
      {
        &quot;id&quot;: 4,
        &quot;name&quot;: &quot;Anjela&quot;
      }
    ]
  }
}">

</div>

## Fetch an object using its primary key

**Example:** Fetch an author using their primary key:

<div class="graphiql" view_only="" query="query {
  authors_by_pk(id: 1) {
    id
    name
  }
}
:response:
{
  &quot;data&quot;: {
    &quot;authors_by_pk&quot;: {
      &quot;id&quot;: 1,
      &quot;name&quot;: &quot;Justin&quot;
    }
  }
}">

</div>

## Fetch value from JSON/JSONB column at particular path

**Example:** Fetch the city and phone number of an author from their
JSONB address column:

<div class="graphiql" view_only="" query="query {
  authors_by_pk(id: 1) {
    id
    name
    address
    city: address(path: &quot;$.city&quot;)
    phone: address(path: &quot;$.phone_numbers.[0]&quot;)
  }
}
:response:
{
  &quot;data&quot;: {
    &quot;authors_by_pk&quot;: {
      &quot;id&quot;: 1,
      &quot;name&quot;: &quot;Justin&quot;,
      &quot;address&quot;: {
        &quot;city&quot;: &quot;Bengaluru&quot;,
        &quot;phone_numbers&quot;: [9090909090, 8080808080]
      },
      &quot;city&quot;: &quot;Bengaluru&quot;,
      &quot;phone&quot;: 9090909090
    }
  }
}">

</div>

See the `API reference <SimpleObject>` for more details.
