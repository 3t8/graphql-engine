## Introduction

Hasura allows connecting to a
[BigQuery](https://cloud.google.com/bigquery) database and building a
GraphQL API based on the database schema.

<div class="admonition">

Supported versions:

Hasura GraphQL engine `v2.0.0-alpha.1` onwards

</div>

## Get Started

To try Hasura with BigQuery, you'll need your own new or existing
BigQuery database.

Here is how you can get started with Hasura and BigQuery:

`Getting Started with BigQuery <database_bigquery_getting_started>`

> 1.  `Hasura Cloud <database_bigquery_cloud>`: You'll need to be able
>     to access your BigQuery database from Hasura Cloud.
> 2.  `Docker <database_bigquery_docker>`: Run Hasura with Docker and
>     then connect your BigQuery database to Hasura.

## Supported features

Hasura currently only supports queries on BigQuery.
