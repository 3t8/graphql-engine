## Introduction

You can fetch a single node or multiple nodes of the same type using a
simple object query.

## Fetch list of objects

**Example:** Fetch a list of authors:

<div class="graphiql" view_only="" query="query {
  authors {
    id
    name
  }
}
:response:
{
  &quot;data&quot;: {
    &quot;authors&quot;: [
      {
        &quot;id&quot;: 1,
        &quot;name&quot;: &quot;Justin&quot;

        &quot;name&quot;: &quot;Beltran&quot;
