## Introduction

Scheduled triggers are used to execute custom business logic at specific
points in time.

alt="Hasura scheduled trigger architecture">

</div>

<div class="admonition">

Supported from

Scheduled triggers are supported from versions `v1.3.0` and above.

</div>

## Types

There are two types of timed events:

-   `Cron triggers <creating_cron_trigger>`: based on a cron schedule.
-   `One-off scheduled events <creating_one_off_scheduled_event>`: based
    on a fixed timestamp.
