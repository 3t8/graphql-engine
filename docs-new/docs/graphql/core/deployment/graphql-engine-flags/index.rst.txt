## Introduction

Hasura supports various flags/env vars for customizing how you are
running the GraphQL engine in your environment.

## Server config

See the `server config reference <server_flag_reference>` for all the
available flags/env vars.

## Use cases

The following are a few configuration use cases:

-   `add-admin-secret`
-   `cli-with-admin-secret`
-   `configure-cors`
-   `console-assets-on-server`
-   `dev-mode`
-   `add-metadata-database`
