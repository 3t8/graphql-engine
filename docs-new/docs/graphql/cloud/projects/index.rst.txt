## Introduction

The `Projects` page shows a list of your projects.

<div class="thumbnail" alt="Projects list" width="1200px">

/img/graphql/cloud/projects/projects-list.png

</div>

For each project, you can do the one of the following actions:

<div class="thumbnail" alt="Project actions" width="860px">

/img/graphql/cloud/projects/project-actions.png

</div>

-   Click `Launch Console` to open the Hasura console in your browser.
    The `Monitoring` tab lets you use the Pro features that Hasura Cloud
    has set up for you.

    <div class="thumbnail" alt="Hasura Console: Monitoring tab"
    width="900px">

    /img/graphql/cloud/metrics/monitoring-tab-overview.png

    </div>

-   Click the gear icon to manage your project

    <div class="thumbnail" alt="General tab" width="900px">

    /img/graphql/cloud/projects/project-details.png

    </div>
