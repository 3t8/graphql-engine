## Introduction

The `Env vars` tab allows setting
`Hasura GraphQL Engine env variables <command-flags>` and adding other
custom env variables as well.

## Adding an env var

Click on the `New Env Var` button and either choose an env var from the
dropdown or add a custom env var.
