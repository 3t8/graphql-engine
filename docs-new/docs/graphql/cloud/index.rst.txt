# Hasura Cloud Documentation

[Hasura
Cloud](https://cloud.hasura.io/signup/?pg=docs&plcmt=body&cta=hasura-cloud&tech=default)
offers hosted [GraphQL Engine](https://github.com/hasura/graphql-engine)
projects with extra features for reliability and security. It includes
all the `core features <core_docs>` of GraphQL Engine, while taking care
of infrastructure concerns, such as the number of instances, cores,
memory, concurrent users, high-availability, realtime monitoring,
caching, tracing, and rate-limiting. It supports both new and existing
databases.
