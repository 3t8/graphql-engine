## How to ...

```bash
for f in docs/graphql/**/**/*.rst; do
  mkdir -pv -- "docs-new/${f%/*}" && touch -- "docs-new/${f%.*}.txt";
  pandoc "$f" -f rst -t gfm -s -o "docs-new/${f%.*}.txt";
done
```
