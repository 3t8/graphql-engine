Hasura.Base.ErrorMessage

Description

Error messages

This module defines a type for user facing error messages.

To construct a value of this type, use toErrorMessage or the IsString interface, the type class ToErrorValue defined in the Hasura.Base.ToErrorValue module, or use the utility functions defined in the Hasura.Base.ErrorValue module.

ErrorMessages can also be composed using the Semigroup interface.

To fail with an error message in monadic code, use the failWithMessage function.
