safeInsertInFieldMap :: (Eq k, Hashable k, Semigroup a1) => InsOrdHashMap k (a2, Maybe a1) -> (k, (a2, Maybe a1)) -> InsOrdHashMap k (a2, Maybe a1) Source #

inserts in field map, if there is already a key, then take a union of the fields inside i.e. for the following graphql query: query MyQuery { test { a b } test { b c } } The final field map of test will have a, b and c
